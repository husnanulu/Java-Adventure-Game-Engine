# Java-Adventure-Game-Engine
A text-based RPG engine demonstrating advanced OOP concepts like Polymorphism, Composition, and Inheritance in Java.

## Advanced Engineering Concepts
* **Polymorphism:** Utilizing method overriding for diverse NPC behaviors (Guards vs. Enemies).
* **Composition:** Creating complex world maps through dynamic Room and Player relationships.
* **Inheritance Hierarchy:** A robust architecture starting from an abstract `Entity` base class.
* **Encapsulation:** Ensuring data security and modularity through strict access control.
